## Requirements
- Vue 3
- Tailwind 3
- laravel 10

## Installation

Clone the repository:
```
git clone repo
```
Install Node dependencies:
```
npm install
```

## Server Run

Change into the laravel project directory:
```
cd memo-card
```
Run the server:
```
php artisan serve
```

Change into the vue project directory:
```
cd memo-card
cd memo
```

Run the dev server:
```
npm run serve
```
